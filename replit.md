# 나스닥 선물 복기 대시보드

## 개요
엑셀 매매 내역을 기반으로 나스닥 선물 거래를 복기하는 대시보드 애플리케이션입니다.

## 기술 스택
- **Frontend**: Streamlit
- **Chart**: Plotly (캔들스틱 차트)
- **Data**: 
  - Yahoo Finance (yfinance) - 기본
  - 한국투자증권 OpenAPI - 선택 가능
- **Database**: SQLite (매매 내역 저장)

## 주요 기능
1. **엑셀 업로드**: 체결내역/청산내역 엑셀 파일을 업로드하여 데이터베이스에 저장
2. **날짜별 조회**: 사이드바에서 날짜를 선택하여 해당 날짜의 매매 내역 조회
3. **차트 표시**: 
   - 캔들 차트: 1분, 3분, 5분, 15분봉 지원
   - 데이터 소스 선택: Yahoo Finance 또는 한국투자증권 API
   - MACD 지표 (선택적 표시)
   - DMI 지표 (선택적 표시)
4. **매매 표시**: 차트 위에 진입→청산 화살표 표시 (빨강=수익, 검정=손실)
5. **시간대**: 시카고 시간(CST) 기준, 한국시간(KST) 병행 표시
6. **원칙 판정 시스템**: 6가지 기준으로 매매 자동 분류
7. **통계**: 원칙 준수율, 원칙매매 승률, 뇌동매매 손실 비용

## 파일 구조
- `app.py`: Streamlit 메인 애플리케이션
- `database.py`: SQLite 데이터베이스 CRUD 모듈
- `kis_api.py`: 한국투자증권 OpenAPI 연동 모듈
- `trade_classifier.py`: 원칙 판정 로직 (6가지 기준 자동 분류)
- `trade_matcher.py`: 체결/청산 매칭 유틸리티 (레거시)
- `trades.db`: SQLite 데이터베이스 파일 (자동 생성)

## 원칙 판정 시스템 (trade_classifier.py)
### 매수 진입 6대 수칙 (3분봉 기준)
1. **33선 필터**: 가격이 33일선 위에 위치
2. **5선 안착**: 가격이 5일선을 상향 돌파 후 안착
3. **MACD 컨펌**: MACD 골든크로스, MACD 본선이 시그널 위
4. **DMI 최종 승인**: DI+가 DI-를 20선 기준으로 상향 돌파 (MACD와 1봉 이내)
5. **시간대**: 17:55~19:00 또는 23:30~06:00 CST
6. **손절 원칙**: -60틱 이내 손절

### 매도 진입 6대 수칙 (3분봉 기준)
1. **33선 필터**: 가격이 33일선 아래에 위치
2. **5선 안착**: 가격이 5일선을 하향 돌파 후 안착
3. **MACD 신호**: MACD 데드크로스
4. **DMI 컨펌**: DI-가 DI+를 20선 기준으로 상향 돌파 (MACD와 1봉 이내)
5. **시간대**: 17:55~19:00 또는 23:30~06:00 CST
6. **손절 준수**: -60틱 이내 손절

### 분류 기준
- **원칙매매**: 5~6개 충족
- **운빨매매**: 3~4개 충족
- **뇌동매매**: 2개 이하 충족

### 기술 지표 설정 (기본값)
- 이동평균선: 5, 33 (3분봉 기준)
- MACD: 7, 20, 6 (단기, 장기, 시그널)
- DMI: 14

## 환경 변수 (Secrets)
- `APP_KEY`: 한국투자증권 API App Key
- `APP_SECRET`: 한국투자증권 API App Secret
- `SESSION_SECRET`: 세션 보안키

## 한국투자증권 API 설정
- **API 엔드포인트**: `/uapi/overseas-futureoption/v1/quotations/inquire-time-futurechartprice`
- **TR_ID**: `HHDFC55020400` (해외선물 분봉조회)
- **종목코드 형식**: `NQH26` (상품코드 + 월코드 + 년도)
- 월코드: F(1월), G(2월), H(3월), J(4월), K(5월), M(6월), N(7월), Q(8월), U(9월), V(10월), X(11월), Z(12월)
- **필수 사전 신청**: CME 거래소 시세 서비스 신청 필요

## 정산일 기준 (한국 증권사 기준)
- **하루 기준**: KST 08:00 ~ 다음날 06:59
- 새벽 07:00 이전 거래는 **전날 정산일**에 포함
- 예시: 2/5 새벽 05:00 거래 → 2/4 정산일로 분류
- 데이터베이스에 `settlement_date` 컬럼으로 저장

## UI 구조 (2026-02-08 업데이트)
- 상단: 4개 통계 카드 (원칙 준수율, 원칙매매 승률, 뇌동매매 손실, 총 손익)
- 중단: 캔들스틱 차트 (전체 너비) + MACD/DMI 서브 차트
- 하단: 2열 가로 배치
  - 왼쪽: 선택된 거래 상세 (진입/청산 정보 + 원칙 판정 6개 기준)
  - 오른쪽: 거래 목록 (요약 통계 포함, 행 클릭 시 차트 강조)
- 거래 선택 시 차트에서 해당 매매 황금색(#FFD700) 하이라이트

## 참고사항
- yfinance 1분봉 데이터는 최근 7일까지만 제공됩니다.
- 한국투자증권 API는 실시간 데이터 조회 가능 (API 신청 필요)
- 차트 범위: KST 08:00 (CST 전일 17:00) ~ CST 16:00 (정산일 기준)
- 매매 시간은 한국 시간(KST)에서 시카고 시간(CST)으로 자동 변환됩니다.
- 선택된 매매는 금색(#FFD700)으로 하이라이트됩니다.
- 차트 Y축 가격 단위: 10 포인트
